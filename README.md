# Hospital Samy Integral

Proyecto CRUD de registro de pacientes desarrollado con PHP y MySQL.

## Archivos principales

- `index.php` - formulario para registrar pacientes.
- `guardar.php` - guarda los datos en la base de datos.
- `listar.php` - muestra todos los pacientes en una tabla con botones Editar / Eliminar.
- `editar.php` - carga un paciente por ID para modificarlo.
- `actualizar.php` - actualiza el registro en la base de datos.
- `eliminar.php` - borra un paciente mediante POST.
- `Conexion.php` - configuración de la conexión a MySQL.
